> 이 프로젝트의 목표는 인스타그램과 같은 소셜 네트워크 서비스를 구축하는 것입니다. 소셜 네트워크 서비스는 모바일 디바이스 사용이 압도적이므로 반응형 웹 구현이 필수적이며, 테마 스위쳐의 적용 역시 필요했습니다.
